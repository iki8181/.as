# Chat Bypasser - SIWO_BAEK Edition

Gabungan script ChatSplitter & ChatBypasser untuk Roblox. Bisa mengirim pesan terlarang dengan karakter Unicode (homoglyph).

## Fitur
- **Input teks bebas** (ketik sendiri)
- **Tombol preset** (Ass, Cock, Bitch, dll)
- **Mode Split** (kirim per kata dengan delay)
- **Toggle Bypass** (on/off)
- **GUI modern** (draggable, minimize, close)

## Cara Pasang
1. Copy script `ChatBypasser.lua`
2. Buka executor (Synapse, Krnl, ScriptWare, dll)
3. Paste dan jalankan
4. GUI akan muncul di layar
